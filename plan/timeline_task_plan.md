## 已做决策
| 决策 | 理由 |
|------|------|
| 时间线模块以 Event 为核心实体 | 事件是所有浏览与关联的中心 |
| 时间表达必须支持不完整和不确定日期 | 历史资料常无法精确到日 |
| 人物、地点、来源应独立建模 | 便于复用、关联和检索 |
| 时间线视图要支持缩放与筛选 | 不同时间跨度下的信息密度不同 |
| 资料来源必须可见 | 档案型系统需要可追溯性 |
