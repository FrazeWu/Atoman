## 已做决策
| 决策 | 理由 |
|------|------|
| 辩论模块采用“结构化论点树”而非扁平评论流 | 便于呈现复杂论证关系 |
| 一个议题可包含多个立场，而不强制只分正反 | 更适合现实复杂议题 |
| 证据引用独立建模 | 有利于追踪信息来源与复用 |
| 投票是立场表态，不直接决定结论真伪 | 避免把人气等同于论证质量 |
| 总结与结论应可迭代更新 | 辩论是演化过程，不是一次性结果 |

## 待讨论
