FEATURE: Unread Discussion Count Badges

DESCRIPTION:
  Visual indicators on album cards showing how many unread discussions exist.
  Users can see at a glance which albums have new discussions without clicking.

BACKEND:
  - File: server/internal/handlers/discussion_handler.go
  - Endpoints: GET /api/albums/:id/discussions/unread-count
              GET /api/songs/:id/discussions/unread-count
  - Returns: {"data": {"unread_count": count}}
  - Query: Counts discussions with read_at IS NULL

DATABASE:
  - File: server/internal/model/revision.go
  - Field: ReadAt *time.Time (indexed)
  - Purpose: Tracks when discussions marked read

FRONTEND:
  - Files: web/src/views/music/HomeView.vue
           web/src/views/music/AlbumDetailView.vue
  - State: discussionCounts Map for caching
  - Rendering: Blue badges showing count
  - Caching: Prevents redundant API calls
